# AI合同风险雷达 - 项目需求

## 项目概述
一个Web应用，帮助用户快速分析合同风险，提供可视化的风险评估报告。使用阿里云百炼平台的大模型能力。

## 核心功能
1. 用户上传合同文件（支持TXT、PDF格式）
2. AI自动分析合同内容，识别风险条款
3. 给出风险评分（0-100分，越高越危险）
   - ≥70分：高风险（红色）
   - 40-69分：中风险（黄色）
   - ≤39分：低风险（绿色）
4. 可视化展示：
   - 雷达图：5个风险维度（权利义务、违约责任、模糊条款、合规风险、缺失条款）
   - 柱状图：高/中/低风险点数量
5. 保存历史分析记录（最多10条，存在session中）
6. 下载分析报告（TXT格式，包含所有分析结果）
7. 侧边栏显示历史记录，点击可查看历史分析

## 技术栈
- **前端/后端框架**：Streamlit（Python Web框架）
- **AI模型**：阿里云百炼 - 通义千问Plus (qwen-plus)
- **API调用**：OpenAI SDK（兼容阿里云DashScope）
- **可视化**：Plotly + Plotly Express
- **PDF解析**：PyPDF2
- **数据处理**：Pandas
- **部署**：Railway / Streamlit Cloud（支持环境变量配置）

## AI分析要求

### 输出格式（JSON）
```json
{
  "total_score": 整数(0-100),
  "risk_points": [
    {
      "clause": "风险条款原文或位置描述",
      "reason": "为什么这是风险，具体说明",
      "severity": "high/mid/low"
    }
  ],
  "dimensions": {
    "权利义务": 0-100,
    "违约责任": 0-100,
    "模糊条款": 0-100,
    "合规风险": 0-100,
    "缺失条款": 0-100
  },
  "summary": "总体风险评估摘要"
}
